# 單字怪物變更紀錄

## 2026-07-01 清爽版 UI 調整

- 將整體視覺從深色厚重 RPG 風格，調整為淡藍白背景、白色卡片、低陰影的清爽風格。
- 保留像素怪物與翻牌遊戲感，但降低深紫底、厚陰影、重外框帶來的壓迫感。
- 調整首頁、字彙包卡片、關卡卡片、戰鬥面板、圖鑑與結果彈窗的色彩、邊框與陰影。
- 將首頁統計文字縮短為「通關 0」「大師 0」，避免手機畫面擁擠。
- 主要修改檔案：
  - `www/style.css`
  - `www/game.js`
- 驗證方式：
  - `node --check www/game.js`
  - `node --check www/data.js`
  - 使用 Chrome headless 擷取 `http://localhost:4174/` 手機尺寸畫面確認顯示正常。

設計方向備註：後續若繼續調整，優先維持明亮、乾淨、低陰影、卡片留白充足的方向；避免回到大面積深紫色與厚重 3D 陰影。
